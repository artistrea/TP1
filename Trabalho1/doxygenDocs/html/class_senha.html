Detailed Description

Senha do usuário.

Regra para a senha ser aceita: Senha deve ter 5 caracteres, dentre eles, deve ter, no mínimo:

  1. Um caracter especial [!#$%&]
  2. Um numero [0-9]
  3. Uma letra [a-zA-Z]
